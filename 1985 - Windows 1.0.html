1985 - Windows 1.0

Windows 1.0 war der erste Schritt von Microsoft in Richtung grafische Benutzeroberflächen. Das System lief als Erweiterung über MS-DOS und war deshalb technisch eingeschränkt. Die Fenster konnten nicht überlappen, sondern lagen nebeneinander, was damals sehr unpraktisch war. Trotzdem brachte Windows 1.0 wichtige Grundlagen: Mausunterstützung, ein einfaches Dateimanagement,Basisprogramme wie Notizblock, Kalender, Rechner und Paint. Die Bedienung basierte auf Icons und Menüs, die an grafische Systeme wie den Macintosh erinnerten. Auf dem Markt blieb Windows 1.0 eher unbedeutend, aber es legte den Grundstein für die spätere Entwicklung der gesamten Windows-Plattform.
