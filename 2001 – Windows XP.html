2001 – Windows XP

Windows XP war besonders, weil Microsoft damit die alte 9x-Reihe endgültig beerdigt und alles auf die NT-Basis gestellt hat. Damit haben selbst normale PC-Nutzer endlich ein stabiles, ordentliches System bekommen. Die neue „Luna“-Oberfläche war ein kompletter Stilwechsel, dazu kamen eine sauberere Nutzerverwaltung, bessere Treiber und insgesamt deutlich weniger Abstürze.

Netzwerk, Multimedia und Internet waren viel enger ins System eingebaut. Gleichzeitig war XP wegen seiner riesigen Verbreitung jahrelang ein Magnet für Schadsoftware, was zu einer langen Reihe von Sicherheitsupdates führte. Trotz dieser Schwachstellen blieb XP extrem populär, weil es robust lief, kaum zickte und in Firmen weit länger im Einsatz blieb, als Microsoft ursprünglich geplant hatte.
