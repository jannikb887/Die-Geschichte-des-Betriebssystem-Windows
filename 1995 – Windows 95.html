1995 – Windows 95

Windows 95 war der größte Umbruch in der frühen Windows-Entwicklung. Startmenü, Taskleiste und ein klarer Desktop-Ansatz haben zum ersten Mal eine Bedienlogik geschaffen, die im Kern bis heute so funktioniert. Technisch lief es auf einem gemischten 16/32-Bit-Unterbau, was Programme spürbar stabiler machte und viele alte Limitierungen der Vorgänger beseitigte. Plug-and-Play kam neu dazu und hat den Umgang mit Hardware erheblich vereinfacht. Für das schnell wachsende Internet brachte Windows 95 grundlegende Netzwerkanbindungen und später per Updates den Internet Explorer. Das System war ein großer Verkaufserfolg und hat Windows endgültig zur dominierenden PC-Plattform der 90er gemacht.
